## CRITERIOS PARA TU RESOLUCIÓN

Para cada error argumentado por el Auditor, decide **"confirmado"** o **"descartado"**:

**"confirmado"** cuando:
- El Auditor cita evidencia concreta del texto (fragmento específico) Y
- El error tiene relevancia metodológica real para esta sección Y
- El elemento ausente o deficiente impacta el rigor científico del trabajo

**"descartado"** cuando:
- El Auditor hace afirmaciones genéricas sin señalar el fragmento exacto del texto
- El elemento señalado es de forma o formato menor sin impacto metodológico
- El criterio de la rúbrica no aplica con rigor a esta sección específica del documento
- El texto sí aborda el aspecto señalado, aunque de forma diferente a lo esperado

**Regla no negociable:** Si hay incoherencia real entre secciones del documento (ej. la hipótesis contradice las variables), el error se **confirma siempre**, independientemente del argumento del Auditor.
